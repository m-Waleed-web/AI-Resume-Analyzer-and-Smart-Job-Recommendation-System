# AI-Resume-Analyzer-and-Smart-Job-Recommendation-System
AI Resume Analyzer and Smart Job Recommendation System Final Year Project
The project is to develop a web system based on Artificial Intelligence that analyzes user resumes and provides intelligent career guidance. The web system will analyze user resumes and provide appropriate guidance based on the information extracted. The user will be able to analyze their resume and make appropriate improvements based on the suggestions provided by the web system. The web system will recommend appropriate jobs based on the skills extracted from the user's resume. The web system will also provide appropriate job opportunities based on user skills. The web system will also have a feature to alert users about available job opportunities based on user skills. The web system will help students and users to effectively improve their resumes and find appropriate jobs.

Technologies Use

HTML, CSS, JavaScript, React.js, Python, NLP libraries spaCy and NLTK, MySQL, GitHub API/Job APIs
